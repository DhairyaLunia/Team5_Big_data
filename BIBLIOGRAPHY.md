# Bibliography & Credits

PersonaPath was built using the following data sources and open-source libraries:

## 📊 Dataset
*   **Yelp Open Dataset:** Provided by Yelp for academic use. [https://www.yelp.com/dataset](https://www.yelp.com/dataset)

## 🐍 Core Technologies
*   **Apache Spark / Databricks:** Distributed data processing and model orchestration.
*   **Gensim LDA:** Latent Dirichlet Allocation (LDA) modeling for topic extraction. [Gensim Documentation](https://radimrehurek.com/gensim/models/ldamodel.html)
*   **VADER Sentiment:** Hutto, C.J. & Gilbert, E.E. (2014). VADER: A Parsimonious Rule-based Model for Sentiment Analysis of Social Media Text. Eighth International Conference on Weblogs and Social Media (ICWSM-14). Ann Arbor, MI, June 2014.
*   **NLTK (Natural Language Toolkit):** Used for tokenization, stopword removal, and text preprocessing.
*   **FAISS (Facebook AI Similarity Search):** Used for high-efficiency vector similarity search. Johnson, J., Douze, M., & Jégou, H. (2019). Billion-scale similarity search with GPUs. IEEE Transactions on Big Data.
*   **Streamlit:** Framework for building the interactive project dashboard.
